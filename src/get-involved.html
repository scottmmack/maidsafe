Get Involved

There are many ways to get involved with the SAFE network.

If you would like to contribute some of your spare computing resource to the network then please visit our farmers page. Farming, analogous to miners in the Bitcoin world, enables all users to earn safecoin (the networkâ€™s crypto currency) in return for supplying storage space, CPU and bandwidth to the SAFE network.

If you are an application developer and are looking to create great applications on the SAFE network then visit our app builder page. Here you will be able to find out more about how the SAFE libraries connect, our in-built revenue model and how to get started.

If you are C++ developer and are interested in helping the community to maintain and progress the underlying network code please visit our core developers page to find out more.

Issues surrounding farming, building and core development are actively discussed in our community run forum https://www.maidsafe.org/. It is a very welcoming and encouraging environment, feel free to dive in and get involved!
